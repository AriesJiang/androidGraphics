## 实现
#### Pixelate Images
1. 选择手机图库图片，进行马赛克处理
2. 滑动进度条，调节马赛克大小

#### Palette Image
1. 获取图片bitmap
2. 使用中位切分法（Median cut）对颜色进行量化处理，得到预先设置的X个主题颜色
3. 将获取到的多个颜色回放的新创建的图像中，得出新的图片
